Steps in the development of the Nonprofit dataset

 

Step

Description

R Scripts

Datasets Produced

1. Compiling Addresses

Generating both Board Members and NPOs addresses datasets.

·   Harmonizing raw dataset variables

·   Removing sensitive information

·   Creating ID + key variables for future manipulation of the dataset

 

 

Step-01-ProcessRawData.rmd

Raw data is compiled into the files:

·   NONPROFIT-ADDRESSES-2014-2019.rds

·   NONPROFIT-2014-2019.rds

·   PEOPLE-2014-2019.rds

2. Identifying POBs and data gaps

Flagging addresses that are POBs, exploring address files and creating some useful variables (pob, add_len, input_address)

 

Step-02-POBsandGaps.rmd

Updated database are saved into:

·   NONPROFIT-2014-2019v2.rds

·   PEOPLE-2014-2019v2.rds

3. Geocoding through the Census Geocode API service

Geocoding addresses through the census service. All addresses are passed through. Step by step walkthrough and troubleshooting.

 

Step-03-CensusGeo.rmd

Input files

·   NPOAddresses_census.rds

·   PPLAddresses_census.rds

 

Raw Results files:

·   NPOAddresses_censusGEO.rds

·   PPLSAddresses_censusGEO.rds

 

These are then integrated into:

·   NONPROFIT-2014-2019v3.rds

·   PEOPLE-2014-2019v3.rds

4. Geocoding through the Google Geocode API service

 

Geocoding addresses through google service. Only non-POB addresses that failed the census geocoding are passed through. Step by step walkthrough and troubleshooting.

Step-04-GoogleGeo.rmd

Input files:

·   NPOAddresses_google.rds

·   PPLAddresses_google.rds

 

Raw Results files:

·   NPOAddresses_googleGEO.rds

·   PPLAddresses_googleGEO.rds

 

These are then integrated into:

·   NONPROFIT-2014-2019v4.rds

·   PEOPLE-2014-2019v4.rds

 

5. Geocoding through Zip and City centroids

Using Zip and City centroids to geocode all addresses that failed through both the census and google geocoding plus all POBs.

Step-05-ZipsandCityGeo.rmd

Results are integrated into:

·   NONPROFIT-2014-2019v5.rds

·   PEOPLE-2014-2019v5.rds

 

6. Adding Census Tracts to dataset

Adding census tracts data for each geocoding method.

Step-06-CensusData.rmd

Results are integrated into:

·   NONPROFIT-2014-2019v6.rds

·   PEOPLE-2014-2019v6.rds

 

7. Data checks and disambiguation

Identifying data gaps, performing data consistency/accuracy checks and determining dataset representativeness.

Step-07-Disambiguation.rmd
